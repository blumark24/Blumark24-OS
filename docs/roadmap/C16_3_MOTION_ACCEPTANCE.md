# C16.3-B Motion Acceptance

## Accepted behavior

- Hover motion is subtle only.
- Focus state has a visible outline token.
- Disabled state has no transition.
- Reduced motion removes transform movement and shortens duration.

## Rejected behavior

- Large movement.
- Audio/video activation.
- Realtime presence activation.
- Meeting invite activation.
- Coordinate or map changes.
- Modal redesign.

## Follow-up

C16.3-C may consume these tokens in UI after review.
