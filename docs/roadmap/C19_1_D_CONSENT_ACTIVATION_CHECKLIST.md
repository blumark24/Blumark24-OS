# C19.1-D — Consent Activation Checklist

Presence consent cannot move from preview to active unless all items are true:

```txt
Tenant scope exists
Office scope exists
Manual status is available
Stop sharing is available
Timeout is available
No audio/video is guaranteed
No silent tracking is guaranteed
User accepted explicitly
Server validation exists
RLS is verified
```
